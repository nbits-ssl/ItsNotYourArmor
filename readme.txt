そんな風なスクリプトなら割と簡単じゃないかな？
まあスクリプトの書き方知らないけどね 


ということで、作りました。
性別が合わない死体から胴装備をルートした場合、元に戻します。
他に設定可能なものとして、スケールと重量、野外はルート可にできたりします。
詳細は「設定」欄に。


## 必要なもの

* SKSE


## 注意事項

* 鎧を剥がないと進まないクエストなどがあった場合には

  stopquest itsnotyourarmor

  でいったんクエストを止め、取得した後に

  startquest itsnotyourarmor

  で再度スタートしてください。

* アンインストールの際もstopquestしてから。


## 設定

MCMはめんどいので全部グローバル変数制御です。
コンソールで「set 変数名 to 値」で変更して下さい。

INYAIndoorOnlyMode (デフォルト1)
0 野外でのルートも不可になります
1 野外では全てのルートが可能です。野外は時間があり仕立て直せる、という想定です。

INYAScaleMode (デフォルト1)
0 身長の相違を無視します
1 死体のキャラの種族身長×死体のキャラ自体の身長の結果がPCと一致しない限り、ルートできません

INYAWeightMode (デフォルト1)
0 重量の相違を無視します
1 重量(weight)がPCと一致しない限り、ルートできません

INYADebugMode (デフォルト1)
0 全てのメッセージ表示を無効にします
1 ルートできなかった際、原因を右上にメッセージ表示します
2 ルートできなかった際、原因をPapyrusログに表示します
